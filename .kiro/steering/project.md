# Project Steering

## Project Overview
This is a web project for CodingCamp by Khanita Anggelina Ruslan. The project is a comprehensive productivity dashboard named **HocusFocus Dashboard** designed to help users manage their focus, tasks, environment, and daily intentions.

## Stack
- HTML (`index.html`)
- CSS (`css/style.css`)
- JavaScript (`js/script.js`)

## Core Features
- **Focus Timer**: A 25-minute Pomodoro countdown timer with Start, Pause, and Reset functionalities to support deep focus sessions.
- **To-Do List**: A dynamic task manager allowing users to add tasks, set custom deadlines (date and time), sort tasks, and track remaining tasks.
- **Dynamic Clock & Greetings**: Real-time digital clock displaying the current time, date, and a personalized greeting system.
- **Daily Quotes**: Displays motivational quotes to keep users inspired.
- **Quick Links (Open Faster)**: A bookmark management section where users can add custom website names and links (e.g., Google, Gmail, Calendar) for quick access, with options to delete them.
- **Spotify Player (Focus Music)**: An embedded audio tool where users can paste their Spotify playlist URL to listen to background music while studying or working.
- **Hocus Space (Personalize)**: A personalization feature allowing users to change and update their display name dynamically on the dashboard greeting.
- **Today's Intention**: A dedicated text input section for users to state and save their main focus or goal for the day.

## Conventions
- Use semantic HTML5 elements.
- Keep CSS organized by component/section.
- Use vanilla JavaScript (no frameworks unless added).
- Validate input data in real-time on the client side (e.g., prevent adding empty tasks or invalid URLs).
- Maintain dynamic UI updates via JavaScript DOM manipulation.
- Indent with 2 spaces.
